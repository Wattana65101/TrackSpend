# ไฟล์รูปภาพโลโก้

กรุณาใส่ไฟล์รูปภาพโลโก้ของคุณที่นี่:

- **ชื่อไฟล์:** `app-logo.png`
- **ตำแหน่ง:** `TrackSpend/assets/images/app-logo.png`

**คำแนะนำ:**
- รูปภาพควรเป็น PNG format
- ขนาดแนะนำ: 200x200 pixels หรือมากกว่า (สำหรับความคมชัด)
- รองรับ transparent background
- ถ้ารูปภาพมีข้อความ "ออมเงินง่าย" อยู่แล้ว component จะแสดงข้อความเพิ่มเติมด้านล่างด้วย

**ตัวอย่าง:**
- ถ้ามีรูปโลโก้พร้อมข้อความ: ใส่รูปภาพที่ `app-logo.png`
- ถ้ามีแค่รูปโลโก้: ใส่รูปภาพที่ `app-logo.png` และ component จะแสดงข้อความ "ออมเงิน" และ "ง่าย" ด้านล่าง
